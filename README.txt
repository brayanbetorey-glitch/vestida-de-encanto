# Vestida de Encanto - Coming Soon

Esta carpeta está lista para subir manualmente a Netlify.

## Contenido
- index.html
- style.css

## Cómo publicarla en Netlify
1. Descomprime este archivo ZIP.
2. Entra a Netlify e inicia sesión.
3. Haz clic en **Add new project**.
4. Selecciona **Deploy manually**.
5. Arrastra la carpeta descomprimida o sus archivos.
6. Espera a que Netlify publique la página.
